# Git verziókezelés

## Helyi repo létrehozása

- helyi repo inicalizálása
    > git init
- ellenőrzés:
    > git status
- előkészítjük a commit-ra (a verzió létrehozására):
    > git add .
- ellenőrzés:
    > git status
- commit:
    > git commit -m "first commit"
- a commit-ok listázása:
    > git log
